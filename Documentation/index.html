Installation

Just download the project to anywhere in your computer and then.

  • Create a database on your environment
  • Open Project in a Code Editor, rename .env.example to .env and modify DB name, username, password to your environment.
  • Open the console and cd your project root directory
  • Run composer install or php composer.phar install
  • An application key can be generated with the command : php artisan key:generate
  • Migrate the database : php artisan migrate
  • Seed Demo Data into database : php artisan db:seed
  • Run the artisan Storage Folder link with public Folder command : php artisan Storage:link
  • Run the artisan serve command : php artisan serve or php -S localhost:8000 -t public
  • Proceed to : http://localhost:8000 for customer view and http://localhost:8000/admin for admin view
